# Installation

Sequentially 
* `composer install`
* `npm install`

# Running

In parallel 
* `php artisan serve`
* `php artisan reverb:start`
* `npm run dev`


# In-game commands 

* `/next [genre]` pulls a new random track
* `/ff` fast forwards in the track
* `/giveup` shows track info
* `/reset` resets the scores
